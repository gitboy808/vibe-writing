# /draft - 生成最终成稿

这是 Vibe Writing System 的成稿阶段快捷命令。

## 执行指令

立即调用以下 Task 工具：

```typescript
Task(
  subagent_type=".claude-plugin:draft-agent",
  prompt="用户希望生成最终成稿，串联所有输出卡片形成完整文章"
)
```

## 使用场景

当你的输出卡片已完成，想要：
- 将所有输出卡片串联成完整文章
- 生成符合认知张力原则的科普文章
- 消除卡片感，形成有机整体

## 前置条件

使用 `/draft` 前，确保：
- 已完成多个输出卡片
- 输出卡片已添加到白板（自动或手动）
- 卡片逻辑关系清晰

## 执行流程

当你使用 `/draft` 时，draft-agent 会：

1. **读取白板**：识别所有输出卡片
2. **分析逻辑顺序**：
   - 检查连接关系和位置
   - 列出逻辑顺序
   - **等待你确认**
3. **串联生成文章**：
   - 优化开头（制造认知张力）
   - 逐句优化内容（悬念、冲突、追问）
   - 优化结尾（引出下一张）
   - 消除卡片感
4. **保存文件**：`项目/[项目名]/最终成稿-[项目名].md`
5. **更新项目信息**：记录成稿状态

## 成稿原则

- ✅ 保留论证链条（大/小逻辑不变）
- ✅ 大胆优化表达（悬念、冲突、追问）
- ✅ 拆分长句、删除啰嗦、突出重点
- ❌ 不复制原文、不改变核心逻辑、不自作主张添加新论点

- `/structure` - 深度分析下一步
