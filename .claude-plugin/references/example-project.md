# 示例项目：Claude和GPT的区别

## 📍 项目位置

**完整路径**: `../项目/Claude和GPT的区别/`

## 🎯 项目概述

这是一个完整的 Vibe Writing System 工作流示例，展示了从零开始创作一篇文章的全过程。

### 项目数据

- **主题**: Claude和GPT的区别（从技术原理到实际使用）
- **知识卡片**: 2张
- **输出卡片**: 4个节点（分2个主题）
- **最终成稿**: 340行，约5000字

## 🔄 工作流展示

### 阶段1：学习对话（learning-agent）

- 用户主题：Claude和GPT的区别
- 初始文档：WebSearch调研生成
- 对话轮次：4轮
- 生成知识卡片：2张

### 阶段2：结构规划（structure-agent）

- 深度分析下一步写什么
- 价值点分析 + 节点拆分
- 生成首版输出卡片：4个节点

### 阶段3：迭代优化（writing-agent）

- 对话迭代
- 自动添加到白板
- 润色优化

### 阶段4：最终成稿（draft-agent）

- 白板卡片串联
- 认知张力优化
- 完整文章生成
