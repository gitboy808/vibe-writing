## 📖 使用建议

### 对于开发者

1. **阅读 `dual-link.md`** 理解知识网络构建机制
2. **参考 `knowledge-card.md`** 学习知识卡片生成规则
3. **查看 `output-card.md`** 了解输出卡片组织原则
4. **研究示例项目** 理解完整工作流

### 对于用户

1. **先看示例项目** 了解 Vibe Writing 的实际效果
2. **参考知识卡片规范** 理解如何组织学习内容
3. **查看输出卡片规范** 了解如何优化表达
